# Прототип кодинг-агента (вроде курсора)
Для проверки, можете посмотреть на файл generated_project/utils.py, там ошибка и попросить агента исправить ошибку командой:
`make coder "fix utils.py"` или `make orchestrator`, после второй он спросит, что сделать.

В работе использовались только самые маленькие модели, поэтому каждый дифф - ровно одна строчка. За счет оркестратора теоретически может достигаться редактирование любого объема (но маленькие модели с этим не справятся).
